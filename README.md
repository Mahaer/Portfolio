# Portfolio

My personal portfolio website built with React and Vite.
Live site: https://undeadtomato.netlify.app

## Setup
```bash
npm install
npm run dev
```

## Notes

- Project data is managed through JSON files in `src/data/`
- Demo videos and images are excluded from this repo and only exist on the deployed site
- The site has a dual-mode privacy system controlled by a URL parameter
